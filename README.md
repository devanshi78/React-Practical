# 🏨 Hotel Management System

A simple and efficient **Hotel Management System** built using **React (Vite)** and **Redux Toolkit**. This project allows users to manage rooms and reservations with a clean UI and simulated backend using JSON Server.

---

## 🚀 Features

- 🔐 User Login (Basic UI)
- 🏨 Room Management
  - View Rooms
  - Room Details
- 📅 Reservation Management
  - Create Reservation
  - View Reservations
- 🔄 State Management using Redux Toolkit
- 🌐 API handling using Axios
- ⚡ Fast development using Vite
- 🗂 Mock backend using JSON Server
